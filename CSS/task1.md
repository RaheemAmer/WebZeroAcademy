# Task

There are many basic different types of selectors.

    - Element Selector
    - Id Selector
    - Class Selector
    - Universal Selector
    - Group Selector

.title {
    Class Selector
}

#nav {
    Id Selector
}


div {
    Element Selector
}


h2 {
    Element Selector
}
